# Echomedic – Smidig prosjekt PRO203

Dette er Los Angeles sin leveranse til prosjekteksamen i PRO203 – Smidig prosjekt høst 2025.  
Vi utvikler et dashboard og rammeverk for informasjonssikkerhet og risikostyring for bedriften Echomedic.

## Struktur

- `echomedic/` – React 18 + Vite + TypeScript dashboard
- `docs/` – Risikoanalyse, SoA, policyer og rapport (kommer)
- `.github/workflows/` – GitHub Actions for lint/build på pull requests

## Kom i gang

```bash
cd echomedic
pnpm install
pnpm run dev
